Description

Guess The Programming Language app is a quizz type app where programmers challenge their knowledge on programming language by guessing these by their logos and short code snippets. You can also:

  • Track your progress while you collect stars and points in the many challenges that are available;
  • Get info about a programming language;
  • Report any incorrect information;
  • Give feedback and play while in offline mode!

It has no authentication system, which means you can start to play right away when you download it!
